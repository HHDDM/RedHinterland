# 红沙腹地｜分层策划案总入口

> 第 1 层主干。这里定义整个区域要让玩家体验什么，并把读者带往六条第 2 层枝干。

## 一句话定位

红沙腹地是一个由绿洲生活、荒漠远行和洞穴调查组成的开放式轻生存解谜区域。玩家通过观察、测量、推理、规划和实验，解决持续出现的现实问题，让水网、商路、生态和聚落生活逐步恢复。

## 核心主题

> 长久并不来自取之不尽，而来自观察、协作、节制和合理的循环设计。

长明灯是这一主题在地下遗迹线中的一个章节，不是区域唯一目标。

## 建议阅读顺序

先读 01、02、03，理解“谁在玩、怎么玩、世界靠什么运转”；再读 04、05，理解“内容如何展开、为何持续投入”；最后读 06，确认第一版的实际制作边界。
