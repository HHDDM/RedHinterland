# B1｜单次任务循环

> 每个任务都遵循“发现异常 → 调查取证 → 提出猜想 → 制定方案 → 准备执行 → 观察结果 → 世界变化”。

[返回核心玩法](README.md) ｜ [返回总入口](../README.md)

## 七个步骤

1. 发现异常：看到农田干枯、水渠断流、商队迟到、风塔停转或动物迁移。
2. 调查取证：与居民交流，获得水量、脚印、影子、温度、震动和路线等证据。
3. 提出猜想：选择调查位置、携带物资或优先修复对象来表达判断。
4. 制定方案：比较不同做法的成本、风险和长期结果。
5. 准备执行：规划路线、水和负重，到现场测量、调整、分配或搭建。
6. 观察结果：世界演示结果，调查手册记录新证据。
7. 世界变化：设施、道路、植物、人物或可接任务发生持续变化。

## 关键边界

- 不以文字选择题代替猜想。
- 不以“答案错误”代替结果反馈。
- 关键流程永远保留重置、回收或替代路径。
